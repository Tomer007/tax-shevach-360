תיעוד אלגוריתם החישוב, הנחות יסוד, ומגבלות — גרסה 1.0

1. סקירה כללית

המערכת מחשבת מס שבח מקרקעין (Capital Gains Tax) לעסקאות מכר דירות מגורים ונכסים בישראל. החישוב מבוסס על חוק מיסוי מקרקעין (שבח ורכישה), תשכ"ג-1963, כולל תיקון 76 (ליניארי מוטב).

תהליך החישוב הבסיסי:

  1. חישוב שווי הרכישה המשוערך (לפי מדד המחירים לצרכן)
  2. חישוב ניכויים משוערכים (שיפוצים, שכ"ט, וכו')
  3. חישוב שבח מקרקעין = שווי מכירה − עלות משוערכת
  4. הפרדה לסכום אינפלציוני ושבח ריאלי
  5. חלוקה ליניארית לפי תקופות (לפני/אחרי 1.1.2014)
  6. השוואת מסלולי מיסוי (ליניארי מוטב, רגיל, פריסה, פטור)
  7. חישוב מס יסף

2. שערוך (אינדקסציה)

2.1 מדד המחירים לצרכן (CPI)

כל סכום (רכישה, ניכויים) משוערך מתאריך הוצאתו ליום המכירה לפי יחס מדדי CPI.

indexed_amount = original_amount × (CPI_sale_year / CPI_acquisition_year)
הנחת יסוד: השערוך מבוצע ברמת שנה (לא חודש). הדבר עלול לסטות ±3-5% עבור רכישות בתחילת/סוף שנה. בפועל, רשות המסים משערכת לפי חודש הרכישה עד החודש שלפני המכירה.

2.2 המרת מטבע

עבור רכישות במט"ח (USD, EUR, GBP), הסכום מומר לשקלים לפי שער בנק ישראל ליום הרכישה.

amount_ils = foreign_amount × BOI_exchange_rate(acquisition_date)

עבור מטבעות היסטוריים:

  • לי"ר ישראלית (ILP) — לפני פברואר 1980: × 0.000001
  • שקל ישן (ILR) — 1980-1985: × 0.001

3. חישוב השבח

3.1 עלות מתואמת

total_cost = Σ(acquisition_indexed) + Σ(deductions_indexed) + betterment_levy − depreciation

3.2 שבח מקרקעין

shevach = sale_amount − total_cost shevach = max(0, shevach) // שבח שלילי = אין מס

3.3 סכום אינפלציוני

הפרש בין הסכום המשוערך לסכום המקורי:

inflationary = Σ(acquisition_ils × (indexation_ratio − 1))

3.4 שבח ריאלי

real_shevach = shevach − inflationary real_shevach = max(0, real_shevach)

4. מסלולי מיסוי

4.1 ליניארי מוטב (תיקון 76)

תנאי זכאות:

  • דירת מגורים מזכה
  • נרכשה לפני 1.1.2014

חישוב:

ratio_after_2014 = days_after_2014 / total_days tax_linear = real_shevach × ratio_after_2014 × 25% // החלק שלפני 1.1.2014 = פטור (0%)

4.3 פריסה (סעיף 48א(ה))

חלוקת השבח הריאלי על פני עד 4 שנים, כאשר כל חלק ממוסה לפי מדרגות מס הכנסה.

spread_per_year = taxable_shevach / num_years year_tax = marginal_tax(spread + other_income) − marginal_tax(other_income) year_tax = min(year_tax, spread × 25%) // תקרה: לא יותר מ-25% total_prisa_tax = Σ(year_tax)

מדרגות מס בפריסה: לפי סעיף 48ב, בתקופת הפריסה השבח ממוסה כהכנסה מעבודה (מדרגות מלאות 10%-50%), ולא כהכנסה פסיבית (רצפה 31%).

נקודות זיכוי: מוחלות לכל המוכרים בפריסה (לא רק מעל גיל 60).

4.4 פטור דירה יחידה — סעיף 49ב(2)

תנאים:

  • דירה יחידה של המוכר
  • בעלות של 18 חודשים לפחות

תקרת פטור (2024): ₪5,008,000

  • מכירה מתחת לתקרה → פטור מלא
  • מכירה מעל התקרה → פטור חלקי
// פטור חלקי (סעיף 49א(א1)): exempt_ratio = ceiling / sale_amount exempt_shevach = real_shevach × exempt_ratio taxable_shevach = real_shevach × (1 − exempt_ratio)

מס נוסף על הכנסה מעל הסף:

6. מס על סכום אינפלציוני

לפי סעיף 48א(ד):

  • סכום אינפלציוני שנצבר לפני 1.1.1994 → מס 10%
  • סכום אינפלציוני שנצבר אחרי 1.1.1994 → פטור
if acquisition_date < 1994-01-01: tax_inflationary = inflationary_amount × 10% else: tax_inflationary = 0

7. תושב חוץ

תושב חוץ ממוסה בשיעור אחיד של 25% על השבח הריאלי, אך:

  • עבור נכס שנרכש לפני 1.1.2014 — רשאי לבחור בליניארי מוטב (כמו תושב ישראל)
  • המערכת בוחרת את הנמוך מבין: מס ליניארי או 25% אחיד

8. פחת

פחת שנוכה או שניתן לנכותו מפחית את בסיס העלות (ובכך מגדיל את השבח):

total_cost = acquisition_indexed + deductions_indexed − depreciation

הפחת אינו משוערך — הוא מופחת ישירות מהעלות המשוערכת.

9. זכויות בנייה (סעיף 49ז)

  • זכויות זניחות (עד ₪100,000) → פטורות לחלוטין
  • פטור כפול — אם שווי הדירה ללא זכויות עד ₪2,428,100: הזכויות פטורות עד ההפרש
  • מעל התקרה → זכויות הבנייה חייבות במס

10. חלוקה בין מוכרים

כאשר יש מספר מוכרים, החישוב מבוצע פר-מוכר לפי חלקו היחסי:

seller_shevach = real_shevach × (share_percent / 100) seller_tax = calculated_tax × (share_percent / 100) mas_yesaf = per_seller (based on seller's other_income)

11. בחירת מסלול מומלץ

המערכת משווה את כל המסלולים ובוחרת את הזול ביותר:

  1. אם זכאי לפטור מלא → פטור 49ב(2)
  2. אם תושב חוץ → 25% אחיד או ליניארי (הנמוך)
  3. אם ליניארי + פריסה זול מליניארי רגיל → ליניארי + פריסה
  4. אם ליניארי זול מרגיל → ליניארי מוטב
  5. אחרת → רגיל
